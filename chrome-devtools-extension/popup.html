DevTools++

Open DevTools and use the DevTools++ panel.

Site Map - Passive host/path tree + DOM scan (links, forms, scripts)
Network - Request monitoring with Initiator call stack + inline source viewer
Intercept - Local MITM proxy: hold/modify/mock requests and responses
Replay - Edit and resend any captured request, auto-diff against original
Global Scope - Single URL scope shared across all features
Shortcut: F12 or Cmd+Opt+I to open DevTools. All features are scoped to the inspected tab.
